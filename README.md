# Unemployment Analysis with Python

## Project Overview
This project analyzes unemployment trends in India using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- State-wise Analysis
- Monthly Trend Analysis
- Covid-19 Impact Analysis
- Data Visualization

## Insights
- Covid-19 increased unemployment rates significantly.
- Some states showed higher unemployment rates.
- Urban and rural unemployment rates differ noticeably.

## Conclusion
This project helped in understanding unemployment trends and improving data analytics skills.
